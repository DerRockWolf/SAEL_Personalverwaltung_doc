    C# - Verwendete Elemente¶

    Dokumentation aller verwendeten Elemente aus C# mitsamt Codebeispielen (Verwendete OOP Elemente)

    Verzweigungen (Conditional Statements)¶

    Mithilfe von Verzweigungen können verschieden Programmabschnitte unter bestimmten Bedingungen ausgeführt werden.

    IF & ELSE¶

    Verzweigungen innerhalb eines Programms werden durch Bedingungen entschieden. Diese Bedingungen können mit IF & ELSE realisiert werden.

    Im Folgegeneden Code-Beispiel wird überprüft, ob die Variable Integer den Wert '8' enhält.
    Wenn diese Bedingung zutrifft, wird der sogenannte IF-Codeblock ausgeführt und dieser gibt anschließend in einer MessageBox den Text "Die Zahl hat den Wert '8'" aus.

    IF-Bedingung¶

    var Integer = 8;

if(Integer == 8)
{
MessageBox.Show("Die Zahl hat den Wert '8' ")
}

Ausgabe:

    "Die Zahl hat den Wert '8' "

    ELSE-Bedingung¶

    else kann benutzt werden, um festzulegen, welcher Code-Block ausgeführt wird, wenn die vorherige IF-Bedingung false ist.

    var Integer = 6;

if(Integer == 6)
{
    MessageBox.Show("Die Zahl hat den Wert '8'.")
}
else
{
    MessageBox.Show("Die Zahl hat nicht den Wert '8'.")
}

Ausgabe:

    "Die Zahl hat nicht den Wert '8'."


    ELSE IF'-Bedingung¶

    Falls eine zusätzliche Bedingung neben der IF-Bedingung hinzugefügt werden soll, kann dies mit Else if realisiert werden.

    var Integer = 1;

if(Integer == 8)
{
    MessageBox.Show("Die Zahl hat den Wert '8'.")
}
else if (Integer == 1)
{
    MessageBox.Show("Die Zahl hat den Wert '1'.")
}

Ausgabe: 

    "Die Zahl hat den Wert '1'."

    Um in den IF & ELSE-Bedingungen z.B mehrere Werte abzufragen, gibt es die Möglichkeit, AND und OR Operatoren zu verwenden. Hier zu können Sie im "Kopf" der IF & ELSE-Bedingung die folgenden Zeichen angeben: | für OR/oder und & für AND/und. Wie Sie die Operatoren im Programmcode implementieren, sehen Sie im nächsten Code-Beispiel.

    Gesamter If-Block-Bedingung¶

    Ein If-Else Codeblock könnte dann zum Beispiel so aussehen:

    var Integer = 4;

if(Integer == 8)
{
    MessageBox.Show("Die Zahl hat den Wert '8'.")
}
else if (Integer == 1 | Integer == 7)
{
    MessageBox.Show("Die Zahl hat entweder den Wert '1' oder '7'.")
}
else 
{
    MessageBox.Show("Die Zahl hat weder den Wert '1' noch den Wert '8'.")
}

Ausgabe: 

    "Die Zahl hat weder die Werte '1', '7' noch den Wert '8'."


    Switch Case¶

    Eine andere Möglichkeit, um Bedingungen zu realisieren, ist das sogenannte Switch Case.
    Der Vorteile von Switch Case gegenüber If-Else sind, dass jenes ab fünf Bedingungen effizierter, schneller und übersichtlicher ist.

    Die Switch Case Syntax ist wie folgend:

    var Integer = 5;


switch(Integer)
{
    case "0":
        MessageBox.Show("Die Zahl hat den Wert '0'." )
        break;
    case "1":
        MessageBox.Show("Die Zahl hat den Wert '1'." )
        break;
    case "2":
        MessageBox.Show("Die Zahl hat den Wert '2'." )
        break;
    case "3":
        MessageBox.Show("Die Zahl hat den Wert '3'." )
        break;
    case "4":
        MessageBox.Show("Die Zahl hat den Wert '4'." )
        break;
    case "5":
        MessageBox.Show("Die Zahl hat den Wert '5'." )
        break;
    case "6":
        MessageBox.Show("Die Zahl hat den Wert '6'." )
        break;

    Ausgabe: 

        "Die Zahl hat den Wert 5"
}



    Ausnahmen (Exceptions)¶

    Try Catch¶

    Um Programmabstürze zu vermeiden, kann Try & Catch (Versuchen & (ab)fangen) verwendet werden. Mit diesem ist es möglich, den Programmfehler und den damit verbunden stop der Anwendung zu verhindern und z. B. eine eigene Fehlerbehandlung zu implementieren.

    Hier Beispiel, wie Try & Catch im Programmcode eingesetzt werden könnte:

    try
{
// Code, der ausgeführt werden soll.
}
catch
{
    // Mögliche Fehlerbehandlung.
}

    Schleifen (Loops)¶

    Schleifen führen einen bestimmten Code-Block so lange aus, bis eine angegebene Bedingung erreicht ist.

    Es gibt drei verschiedene Arten von Schleifen:

    • Nicht abweisende Schleifen.
      Beispiel: die do-while-Schleife.

    • Abweisende Schleifen.
      Beispiel: die while-Schleife.

    • Zählschleifen.
      Beispiel: die for-Schleife.

    do while-Schleifen¶

    Die do while ist eine fußgesteuerte Schleife, dass bedeutet, dass die angegebene Bedingung erst am Ende ("Fuß") der Schleife überprüft wird. 

    var Integer = 1;

do 
{
    Integer++;
}
while (integer < 10)

    Das Programm wird die Schleife verlassen, nachdem die Variable Integer den Wert '10' erreicht hat und dadurch die Bedingung nicht mehr erfüllt wird.

    while-Schleifen¶

    var Integer = 1;

while (integer < 10)
{
    Integer++;
}


    Die Schleife wird verlassen, nachdem die Variable den Wert '10' erreicht hat und dadurch die Bedingung nicht mehr wahr ist ('10' ist nicht kleiner als '10' )

    for-Schleifen¶

    Die for ist eine Zählschleife. Diese eignet sich gut, um z. B. Variablen hoch- oder herunter zu zählen. Im Schleifenkopf wird die zu überprüfende Variable initialisiert, die Bedingung angegeben und zusätzlich wird festgelegt, was in den Schleifendurchläufen mit der Variable geschieht (z. B. der Wert wird jeweils um 1 erhöht). 

    for (var integer = 1; integer < 10; integer++) 
{
    // Hier kann z. B. der derzeitige Wert der Variable ausgegeben werden.

Console.WriteLine(integer);

}

    Nachdem die Variable den Wert '10' erreicht hat, wird der Schleifen-Codeblock nicht mehr ausgeführt und dadurch ist die letzte Zahl, die ausgegeben wird der Integer '9'.

    Datenstrukturen¶

    Datenstrukturen ermöglichen das strukturierte Speicherung und Abrufen von Daten.

    Arrays¶

    Arrays können verwendet werden, um mehrere Variablenwerte in einer einzelnen Variable zu speichern, um damit zu umgehen, dass jeder Wert einzeln deklariert werden muss.

    So kann man zum Beispiel einen (eindimensionalen) Array mit der Größe '5' zu deklarieren und initialisieren:

    int[] integer = new int [5];

    So könnte man einem Array einzelne Werte zuweisen:

    integer[0] = 54;
integer[1] = 13;

    Es ist auch möglich, dem Array seine Werte direkt bei der Deklarierung zuzuweisen:

    int[] integer = {45, 3, 54, 13, 43};

    Mehrdimensionale Arrays¶

    Es gibt auch die Möglichkeit, einen mehrdimensionalen Array zu erstellen. Mehrdimensional bedeutet, dass in dem Array nun in den einzelnen Spalten mehrere Werte eingespeichert werden können.

    Um einen Mehrdimensionalen Array zu erzeugen, muss lediglich bei der Initialisierung in den ersten eckigen Klammern ein Komma gesetzt werden.

    Hier ein Code-Beispiel:

    int[,] integer = new int [5,3];

    Der Array hat nun sechs Spalten in denen bis zu vier Werte eingespeichet werden können.

    Warum sechs Spalten und vier Werte?

    Antwort:

    Das liegt daran, dass Arrays nullbasierend sind. Das bedeutet, dass der Array den ersten Variablenwert mit dem Index '0' einspeichert. Deshalb ist der reelle Wert des Arrays immer um einen Integer erhöht.

    So könnte dann zum Beispiel ein Wert in einem mehrdimensionalen Array eingespeichert werden:

    integer[0,0] = 45;
integer[0,1] = 21;

    Um die Größe eines Array flexibel zu gestalten, gibt es die Möglichkeit, ihn mithilfe von Variablen zu deklarieren und zusätzlich mit Ihnen das Einspeichern von einzelnen Werten zu übernehmen.

    int a = 78; 
int b = 43; 
int c = 0;
int d = 1;

int [,] integer = [a,b];

integer [c,d] = 13;

    Die Variablen können dann innerhalb des Programmcodes verändert werden und dadurch die Größe des Array und die eingespeicherten Werte beeinflussen.

    Listen¶

    In Listen können wie in einem Array mehrere Variablenwerte eingespeichert werden.

    Sie kann wie folgend deklariert werden:

    List<string> Buchstaben = new List<string>()

    Um einer Liste einen Wert zuzuweisen, kann man dies z. B. wie in diesem Code-Beispiel tun:

    Buchstaben.Add("A")
Buchstaben.Add("B")

    So können Sie die Werte einer Liste auszugeben:

    Console.WriteLine(Buchstaben[0])

Ausgabe: "A"

//Es wird der Wert ausgegeben, welcher als Erstes der Liste hinzugefügt wurde.

    Methoden/Funktionen¶

    Grundlagen¶

    Methoden werden verwendet, um bestimmte Aktionen im Programmcode auszuführen, Sie werden auch als Funktionen bezeichnet.

    Der Vorteil an Methoden ist, dass Sie mehrmals verwendet werden können. Dadurch kann vermieden werden, den gleichen Code mehrmals zu verfassen und stattdessen eine Methode, die auch diesen Code enthält, aufzurufen.

    Info

    Es gibt in C# auch vorprogrammierte Methoden. Bekannte Vertreter hiervon sind:

    Console.WriteLine oder MessageBox.Show

    In diesem Code-Beispiel wird ihnen nun gezeigt, wie Sie eine eigene Methode erstellen können. 

        void TestMethode()
    {
        // Programmcode der in der Methode ausgeführt werden soll.
    }

    Um diese Methode anschließend im Programm aufzurufen, gehen Sie wie folgend vor:

    static void Main (string[] args) 
{
    TestMethode();
}

void TestMethode() 
{
    MessageBox.Show("das ist ein Test!")
}

Ausgabe: "Das ist ein Test"

    Methoden mit Argumenten¶

    Es gibt die Möglichkeit, einer Methode aus dem Programmcode heraus Information mitzugeben. In C# werden diese Information Argumente genannt. Diese Argumente können beim Aufruf der Methode in den Klammern mitgegeben werden und anschließend wird im Kopf der Methode angeben, als welchen Datentyp und mit welchem Namen die Methode das mitgegebene Argument verarbeiten soll.

    Hier ein Code-Beispiel zu Argumenten:

        static void Main (string[] args) 
    {
        TestMethode("Hallo");
    }
    void TestMethode(string Begrüßung) 
    {
        MessageBox.Show(Begrüßung + "das ist ein Test!")
    }

    Ausgabe: "Hallo das ist ein Test!"

    In diesem Beispiel wird der Methode die Zeichenfolge "Hallo" als Argument übergeben. Die Methode verarbeitet das Argument als string und gibt diesem den Namen Begrüßung. Anschließend werden der string Begrüßung und die Zeichenfolge "das ist ein Test!" in einer MessageBox ausgegeben.

    Es natürlich auch möglich, mehrere Argumente anzugeben, diese müssen jedoch mit einem Komma getrennt werden:

          static void Main (string[] args) 
    {
        TestMethode("Hallo", "Max");
    }
    void TestMethode(string Begrüßung, string Name) 
    {
        MessageBox.Show(Begrüßung + Name + "das ist ein Test!")
    }
    Ausgabe: "Hallo Max das ist ein Test!"

    Argumente per Referenz übergeben¶

    Wenn Sie einer Methode eine Variable per Argument übergeben, werden diese Variablen sozusagen kopiert und anschließend im Methoden-Cope verwendet. Falls diese "kopierten" Variablen in der Methode verändert werden, hat dies keine Auswirkung auf das Original. Diese Variable bleibt unverändert. Wenn Sie jedoch möchten, dass die Veränderung in der Methode Auswirkungen auf das Original zeigt, können Sie die per Referenz und dem Schlüsselwort ref tun.

    Hier ein Code-Beispiel:

        int x = 5;
    int y = 2;

    static void Main (string[] args) 
    {
        TestMethode(ref x, ref y);
        MessageBox.Show(x,y)
    }
     /* Die Variablen  'x' und 'y' werden der Methode 
    als Variable 'a' und 'b' übergeben.
    */
    void TestMethode(ref int a, ref int b) 
    {
        x = 19;
        y = 3;
    }

    Ausgabe: x = 19 "und" y = 3;

        int x = 5;
    int y = 2;

    static void Main (string[] args) 
    {
        TestMethode(x,y);
        MessageBox.Show(x,y)
    }
    /* Die Variablen  'x' und 'y' werden der Methode 
    als Variable 'a' und 'b' übergeben.
    */
    void TestMethode( int a, int b) 
    {
        a = 19;
        b = 3;
    }

    Ausgabe: x = 5 "und" y = 2;

    Methoden mit Rückgabewert¶

    Die vorherigen Methode-Beispiele hatten alle den Datentyp void. Mit diesem ist es nur indirekt möglich, Wert zurückzugeben. Wenn die Methode jedoch mit einem Datentyp wie zum Beispiel int oder double deklariert wird, ist es über das Schlüsselwort return möglich, einen Rückgabewert zu erhalten.

    Wie dies möglich ist, sehen Sie im folgenden Code-Beispiel:

        int x = 8;
    int y = 5;
    int z = GrößterWert(x,y);

    static void Main(string[] args)
    {
        GrößterWert(x,y);
        MessageBox.Show("Die größte Zahl ist: " + z);
    }

    static int GrößterWert(int a, int b) 
    {
        if (a > b)
        {
            return a;
        }
        else
        {
            return b;
        }
    }

   Ausgabe: "Die größte Zahl ist: 8"

    In diesem Beispiel werden der Methode GrößterWert die beiden Variablen x und y übergeben. Die Methode ermittelt anschließend, welcher der beiden Werte größer ist und übergibt je nachdem den größten Wert. Dieser wird darauffolgend als Variable z in einer MessageBox ausgegeben.

    FileWriter/FileReader¶

    Mit Hilfe des FileWriter und des FileReader ist es möglich, Textdateien zu erstellen und diese bei Start des Programms einzulesen.

    File Writer¶

    Mit dem File Writer ist es möglich, einen Dateipfad für eine Textdatei anzugeben und dann den Inhalt der Datei zu bestimmen.

    Hier folgendes Code-Beispiel:

    string DateiPfad = "C:\tmp\Test.txt";
string Inhalt = "Test";
File.WriteAllText(DateiPfad , Inhalt);

    In diesem Beispiel wird als Pfad "C:\tmp\Test.txt" angegeben. In die Textdatei wird dann ein String mit der Zeichenfolge "Test" hineingeschrieben.

    File Reader¶

    Mit Hilfe des File Reader ist es im Anschluss möglich, die gespeicherte Textdatei wieder im Programm einzulesen und zum Beispiel einer Variable zuzuweisen.

    string DateiPfad = "C:\tmp\Test.txt";
string Inhalt = "";
Inhalt = File.ReadAllText(DateiPfad);
